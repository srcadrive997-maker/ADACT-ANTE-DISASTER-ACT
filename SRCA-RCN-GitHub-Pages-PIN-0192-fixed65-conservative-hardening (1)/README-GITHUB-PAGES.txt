رفع المنصة على GitHub Pages مع بقاء PIN 0192

1) أنشئ Repository جديدا على GitHub.
2) ارفع محتويات هذا المجلد كما هي، وليس ملف ZIP داخل المستودع.
3) تأكد أن الصفحة الرئيسية اسمها index.html.
4) من Settings > Pages:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
5) احفظ الإعدادات وانتظر اكتمال النشر.
6) سيفتح الموقع بالرابط الذي يمنحه GitHub Pages.
7) كود الدخول الحالي داخل المنصة سيبقى: 0192

تنبيه مهم:
شاشة PIN ستظل تعمل، لكنها حماية واجهة أمامية فقط. إذا كان المستودع عاما أو الموقع عاما، فيمكن تقنيا الوصول إلى كود الصفحة ورؤية رقم PIN. للحماية الحقيقية يلزم نظام دخول على خادم أو استضافة خاصة.
